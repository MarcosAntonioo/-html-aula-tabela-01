a. Tabela ( ) Podem conter todos os tipos de elementos HTML:Textos, imagens,
listas, outra tabelas, etc.
b. Linha ( ) Por padrão está em negrito e centralizado.
c. Coluna ( ) Em uma tabela é utilizada a tag tr.
d. Cabeçalho ( ) Por padrão é regular e alinhado à esquerda.
e. Células ( ) São compostas por linhas e colunas.
